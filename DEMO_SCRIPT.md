# Learning Arc — 3-minute demo

## 0:00–0:30 — Problem

“Learning platforms tell us what we completed. Productivity tools tell us how long we worked. Neither tells a self-learner whether they are becoming capable of doing the work independently. Learning Arc makes that growth visible—without signup or surveillance.”

## 0:30–1:30 — Focus → reflection → AI

Show **Today**, then **Focus**. Pick Building, type “Async JavaScript”, add an intent, and point out the timestamp-based timer. Use the clearly labelled developer test control only for the demo. Complete the short session.

In the reflection screen, enter: “I built a small API request without following the lesson.” Choose “Mostly independently,” then press **Analyze my progress**. Explain: “The session is saved first. Gemini interprets structured evidence only after I ask; if quota fails, the evidence and analytics remain.” Show the resulting event on Today.

## 1:30–2:20 — Journey + Learning Intelligence

Open **Settings → Load demo journey** and confirm (or start with it already loaded). Go to **Journey**. “This isn’t a generic contribution chart: each active day opens the actual learning evidence. The skill evolution lane reflects repeated sessions and application, not certifications.” Click an active day.

Open **Learning Intelligence**. “Counts and balance are calculated locally. AI is asked to interpret those facts—not invent them. The key story here is a shift from guided lessons to independent API work, plus one concrete next move.”

## 2:20–2:50 — Proof of Learning

Open **Proof of Learning**. “GitHub shows what you built. Learning Arc can show how you became capable of building it: focused evidence, active streak, balance, skills, and an honest AI interpretation. It can be copied or printed as a portfolio-ready local record.”

## 2:50–3:00 — Conclusion

“Learning Arc doesn’t just measure completion or time. It helps learners understand how they are learning—and choose the next action that turns knowledge into independent capability.”
